# Smart Vocabulary Builder

A full-stack vocabulary learning application built using React, Node.js, Express.js, and MongoDB Atlas.

## Live Demo
https://smart-vocab-builder.vercel.app

## Features
- Add new words
- Fetch meanings using Dictionary API
- Store words in MongoDB Atlas
- View recently added words
- Responsive UI

## Tech Stack
- React.js
- Tailwind CSS
- Axios
- Node.js
- Express.js
- MongoDB Atlas

## Deployment
- Frontend: Vercel
- Backend: Render
